# 小红书平台 Skill

> 适用场景：小红书图文笔记  
> 核心目标：推送给正确人群 + 实现商业转化，不以追求爆文为唯一目标

---

## 一、标题规则

### 1.1 标题构成要素

一个有效标题必须包含以下至少 2-3 个元素：

| 元素 | 说明 | 举例 |
|------|------|------|
| **人群词** | 锚定目标受众 | 黄黑皮、30+女性、考研党、宝妈、打工狗、爸妈|
| **症状词** | 描述用户痛点 | 脱发、肝损伤、注意力不集中、疲劳、长痘、脱发 |
| **热点词** | 蹭当前搜索热度 | 维A酸、青少年抑郁、618大促 |
| **数量词** | 增加真实感和可信度 | 吃了1周、30天、才十几块 |
| **品类词** | 锁定搜索意图 | 鱼油、维生素D3、保健品 |
| **效果词** | 直接呈现具体结果 | 震惊了、真的媲美、反馈 |

### 1.2 标题目标分类

**转化型标题**（锁定已有购买意向人群，转化率高但流量小）
- 结构：品牌词 + 产品词 + 使用反馈
- 示例：`金凯撒鱼油吃了1周反馈`
- 适用：已有认知、犹豫未下单的精准人群

**拉新型标题**（扩大人群认知，流量大但需要更强钩子）

| 类型 | 结构 | 示例 |
|------|------|------|
| 效果吸引 | 人群词 + 产品 + 时间 + 结果 | `黄黑皮吃鱼油30天真的震惊了！` |
| 恐吓吸引 | 警告 + 产品 + 风险后果 | `不要再吃鱼油了小心肝损伤！` |
| 海外背书 | 地域权威 + 产品推荐 | `大家一定要去买意大利药店的这款鱼油` |
| 类目对比 | 产品 + 媲美 + 知名参照物 | `这个东西真的媲美维A酸软膏！` |
| 年龄认同 | 年龄段 + 痛点疑问 | `30+女性到底该吃什么保健品？` |

## 二、开头钩子规则

> 核心原则：让用户一眼 get 文章主题，场景/情绪类可100-300字，其他类型控制在150字以内

### 2.1 颠覆性观点
- 必须基于真实存在的研究或近期平台搜索量上升的话题
- 必须与用户需求匹配
- 示例：
  - `你以前的维生素D3都吃错了！`
  - `新研究：维生素D3竟然还可以抗衰？`
  - `5000IU维生素D3会中毒`

### 2.2 具体场景（重点）
- ❌ 错误写法：堆砌多个场景（经常吃外卖、熬夜、压力大）→ 共鸣弱
- ✅ 正确写法：选定**一个场景**，深挖细节，大量使用数量词，还原真实感

  **示例（脱发场景）：**
  > 马上要12月份考研了，我从10月份开始就发现发根开始有松动！一开始就是洗头的时候掉的多大概就是五六十根的样子，现在是每天早晚梳头密密麻麻的一百多根，我真的哭了，本身就考研，现在还脱发真的更焦虑了

  **关键技巧：**
  - 锁定一个具体时间节点
  - 用数量词描述程度变化（五六十根→一百多根）
  - 挖掘该场景下的真实心态和情绪

### 2.3 具体情绪
- 直接用口语化表达，不需要科普
- 适用于情绪价值强的产品（洗护、香氛、亲子类）
- 示例：
  > 强推一个无人问津的才十几块！爱上洗手了555在家我只要路过就想洗一次手 洗完立马猛嗅一口 超好闻木棉皂香！！

### 2.4 方法论
- 适用于攻略类内容
- 用户注意力短暂，先总结后发散效果更好
- 结构：直接给出核心结论 → 再展开说明

### 2.5 权威背书
- 用权威人物/机构的原话直接作为开头
- 适用：医学、营养、科技类产品

### 2.6 社会热点
- 将产品功效与当前热点事件关联
- 示例：用张雪峰猝死热点引出辅酶Q10改善心脏健康

## 三、正文结构规则

### 3.1 测评文
```
开头钩子
→ 定义：什么样的产品才是好的（建立评判标准）
→ 对比：成分 / 含量 / 价格横向对比多款产品
   - 自家产品：只说优点
   - 竞品：抓缺点做避雷
→ 结论：最终选择了哪款 + 理由
```

### 3.2 广告文
```
开头钩子
→ 产品为什么能解决上述问题（机制说明）
→ 为什么要选我家产品（差异化优势）
→ 当前优惠机制 or 使用感受
```

### 3.3 体验文（软广）
```
选定一个具体场景 → 深挖细节和情绪
→ 中间自然植入：我买了这款产品
→ 为什么买这款（朋友推荐 / 营养师推荐）
→ 产品效果（用用户语言描述，不过度介绍）
```
> ⚠️ 体验文核心：内容真实感优先，产品介绍要软，用用户语言而非品牌语言

### 3.4 攻略文
```
开头钩子
→ 哪些方法可以解决问题（给方法论）
→ 推荐产品（自然引入）
```

## 四、结尾与评论引导规则

### 4.1 结尾写法（三选一或组合）
- **总结好处**：直接提炼产品核心价值
- **信任背书**：我已推荐给X个朋友 / 已复购X次
- **优惠逼单**：当前活动价 / 限时优惠 / 赠品机制

### 4.2 评论区引导（按转化漏斗设计）

| 评论类型 | 目的 | 示例 |
|----------|------|------|
| 对号入座型 | 引导互动，扩大覆盖人群 | `我有XX问题，可以用这款吗？` |
| 方法论型 | 提供价值，促进信任 | `我XX问题需要用多久才有效果？` |
| 收割型 | 直接引导购买 | `在哪里买？` |
| 真实反馈型 | 增加可信度（可配合托） | `我也买了，效果XXXX` |

## 五、标签选择规则

1. 从标题和正文中提取核心关键词
2. 每个关键词在小红书搜索，优先选择**该词下热度最高的话题**
3. 搭配结构：
   - 大热度话题（引流）× 2-3个
   - 精准垂类话题（转化）× 3-5个
   - 品牌/产品词话题 × 1-2个

## 六、图片与排版规则

### 6.1 排版原则
- 一段话、甚至一个重点，单独一行，方便用户快速扫读
- 适当添加匹配内容的 emoji，增加视觉节奏
- 除真实体验文外，其他类型笔记每段不超过 5 行

### 6.2 字数建议

| 笔记类型 | 字数建议 | 说明 |
|----------|----------|------|
| 广告文 | 讲清楚即可 | 不追求字数，说完即止 |
| 体验文 | 尽量精简 | 让用户快速看懂产品是干什么的 |
| 测评文 | 讲清楚即可 | 对比清晰比字数多更重要 |
| 攻略文 | 尽量丰富 | 可用满1000字符上限 |

> 小红书字符上限 1000，攻略文之外的内容不需要刻意凑字数

## 七、与抖音内容互转规则

### 7.1 不变的部分
- 锚定的核心人群
- 锚定的核心场景
- 产品的核心卖点

### 7.2 必须改变的部分

| 维度 | 小红书 | 转抖音后 |
|------|--------|----------|
| 语境 | 图文细读，用户会停留 | 视频快消，用户随时划走 |
| 文字量 | 可以较多，铺陈场景 | 大幅压缩，只留核心 |
| 信息顺序 | 钩子→铺垫→结论 | 结果和重点必须前置 |
| 语言风格 | 小红书人群语境 | 换成抖音人群语境露出 |

### 7.3 转化口诀
> 小红书是"讲故事"，抖音是"给答案"  
> 红薯能写300字铺垫的，抖音只给你10秒说完结论

### 7.4 具体操作
直接参考 [抖音 Skill →](douyin.md) 的结构规则进行内容重组
